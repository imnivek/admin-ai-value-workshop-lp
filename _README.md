## 設計方向

- 色彩：暖米色背景、深森林綠主色、暖金與珊瑚點綴。
- 調性：高質感、柔和、專業、女性職場友善，但不做過度裝飾。
- 素材：不依賴外部圖片下載，所有主要視覺都在 HTML 內完成，適合直接預覽與後續改版。
- 文案：從「秘書 AI 課」調整成「行政工作價值整理」，更符合 TA 對履歷、主管溝通、資料整理、轉職與補助優惠的購買動機。
